isol8 ships with optimized Docker images for 5 major runtimes.

## Python

The default runtime for inline code.

<Tabs>
  <Tab title="Overview">
    - **Image**: `isol8:python`
    - **Base**: Alpine 3.21
    - **Extension**: `.py`
    - **Packages**: `pip install` (installs to `/sandbox/.local`)
  </Tab>
  <Tab title="Example">
    ```bash
    isol8 run -e "import sys; print(sys.version)" --runtime python
    ```
  </Tab>
</Tabs>

### Dockerfile
```dockerfile
FROM base AS python
RUN apk add --no-cache python3 py3-pip
CMD ["python3"]
```

## Node.js

Stable LTS environment.

<Tabs>
  <Tab title="Overview">
    - **Image**: `isol8:node`
    - **Base**: Alpine 3.21
    - **Extension**: `.js`, `.mjs`, `.cjs`
    - **Packages**: `npm install` (installs to `/sandbox/.npm-global`)
  </Tab>
  <Tab title="Example">
    ```bash
    isol8 run -e "console.log(process.version)" --runtime node
    ```
  </Tab>
</Tabs>

### Dockerfile
```dockerfile
FROM base AS node
RUN apk add --no-cache nodejs npm
CMD ["node"]
```

## Bun

Fast startup, native TypeScript support.

<Tabs>
  <Tab title="Overview">
    - **Image**: `isol8:bun`
    - **Base**: Alpine 3.21
    - **Extension**: `.ts`, `.tsx`
    - **Packages**: `bun install -g` (installs to `/sandbox/.bun-global`)
  </Tab>
  <Tab title="Example">
    ```bash
    isol8 run -e "console.log(Bun.version)" --runtime bun
    ```
  </Tab>
</Tabs>

### Dockerfile
```dockerfile
FROM base AS bun
RUN apk add --no-cache bash unzip libstdc++ libgcc 
    && curl -fsSL https://bun.sh/install | bash
CMD ["bun"]
```

## Deno

Secure-by-default runtime.

<Tabs>
  <Tab title="Overview">
    - **Image**: `isol8:deno`
    - **Base**: `denoland/deno:alpine` (Custom base)
    - **Extension**: `.mts`
    - **Packages**: `deno cache` (URL imports)
    - **Caveat**: No inline code support (`-e`). Must run files.
  </Tab>
  <Tab title="Example">
    ```bash
    isol8 run script.mts
    ```
  </Tab>
</Tabs>

### Dockerfile
```dockerfile
FROM denoland/deno:alpine AS deno
# Re-installs proxy tools since it doesn't use the shared base
RUN apk add --no-cache tini curl ca-certificates
```

## Bash

For system automation and piping.

<Tabs>
  <Tab title="Overview">
    - **Image**: `isol8:bash`
    - **Base**: Alpine 3.21
    - **Extension**: `.sh`
    - **Packages**: `apk add` (runs as root during install step)
  </Tab>
  <Tab title="Example">
    ```bash
    isol8 run -e "echo $SHELL" --runtime bash
    ```
  </Tab>
</Tabs>

### Dockerfile
```dockerfile
FROM base AS bash
RUN apk add --no-cache bash
CMD ["bash"]
```
