# What is isol8?

isol8 is a secure code execution engine that runs untrusted code inside Docker containers with strict resource limits, network controls, and output sanitization. It is designed for AI agents, chatbots, and any application that needs to safely execute user-provided code.

## How It Works

1. You submit code with a runtime (Python, Node, etc.) and optional configuration
2. isol8 runs it inside a locked-down Docker container
3. Output is collected, secrets are masked, and results are returned
4. The container is destroyed (ephemeral) or preserved (persistent)

## Supported Runtimes

| Runtime | Image | File Extension | Package Manager |
|:--------|:------|:---------------|:----------------|
| Python 3.x | `isol8:python` | `.py` | pip |
| Node.js LTS | `isol8:node` | `.js` | npm |
| Bun | `isol8:bun` | `.ts` | bun |
| Deno | `isol8:deno` | `.mts` | deno cache |
| Bash (Alpine) | `isol8:bash` | `.sh` | apk |

<Note>
  `.ts` files default to Bun. Use `--runtime deno` or the `.mts` extension for Deno.
</Note>
